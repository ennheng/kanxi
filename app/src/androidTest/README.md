# 仪器测试范围

这些测试只依赖本地 Compose、Room 和确定性的完整 B 站 URL，不把 B 站在线可用性作为自动化断言。

已覆盖：

- 空目录、分类入口、分类列表和直接播放入口的语义与回调；
- 本地别名搜索、详情页收藏及收藏列表；
- 显式 `ACTION_SEND text/plain` 进入应用、导航到编辑器，以及标题/链接预填与单次消费；
- 手工粘贴完整链接后的验证、保存，以及已有条目编辑；
- 家人管理的编辑路由、删除确认、取消和实际删除；
- 不安全播放器 URL 的关闭失败路径，以及规范 B 站原页面外部兜底。

以下行为不适合作为稳定的 CI 仪器测试：

- 不同厂商系统 Sharesheet 中“看戏”入口的展示与点击（测试会直接向 Activity 发送等价 Intent）；
- `b23.tv` 线上重定向和 B 站播放器实时可用性；
- WebView 断网、渲染进程崩溃、官方播放器全屏与屏幕旋转。

这些项目依赖系统组件、WebView 版本、真实网络或 B 站在线状态，应按 README 的真机清单验收。
